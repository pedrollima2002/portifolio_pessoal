# Portfólio Pessoal

Projeto de portfólio pessoal desenvolvido com HTML e CSS para apresentar minhas habilidades, projetos e informações de contato.

## Tecnologias Utilizadas
- HTML5
- CSS3

## Funcionalidades
- Menu de navegação
- Seção sobre mim
- Área de skills
- Cards de projetos
- Formulário de contato
- Layout responsivo
- Animações visuais
